人數多空比

定義

在交易市場中，多頭（看漲）和空頭（看跌）頭寸的比例，人數多空比 = 多頭人數 / 空頭人數

交易所中的多頭持倉和空頭持倉是相互對等的，1比1的關係。 如果總倉位價值相等，而持有人數不同，那就說明持有人數多的一方人均倉位價值較小，以散戶為主，另一方以大戶和機構為主。

假如目前人數多空比為 2 代表的意義是 多頭人數 / 空頭人數 = 2， 持有多單的人數是持有空單人數的 2 倍， 多方人多，人均持倉量更少，散戶佔比多。

多頭持倉（Long Positions）：投資者下的買入合約，表明他們預期未來價格會上升。
空頭持倉（Short Positions）：投資者下的賣出合約，表明他們預期未來價格會下降。

依經驗絕大多數的情況下，散戶都是習慣性的做多為主，對手盤可能是做市商、機構與大戶。

其它多空比

除了人數多空比之外，交易所還有提供大戶的多空比資料，大戶為交易所內保證金餘額排名前20%的用戶

1、大户帳戶數多空比:

大戶的淨持倉多頭人數 / 大戶的空頭賬戶人數

2、大户帳戶 持倉量多空比:

大戶的總持倉量多頭 / 大戶的總持倉量空頭

大戶多空比資料參考就好，下面以都是以 「人數多空比」做為探討，是人數之比例，並不代表市場多方與空方資金力量的對比。

多空持倉人數比，實際上反應的是市場散戶的情緒

觀察情緒變化

人數多空比是重要的散戶情緒觀察指標，依定義而言

人數多空比高 >3 代表散戶非常樂觀在看多，人數是做空的3倍。
人數多空比低 <0.8 代表散戶非常悲觀在看空，人數是做空的0.8倍。

所以 人數多空比降低▼ 代表市場看多▲ ， 人數多空比上升▲ 代表市場看空▼

但現實的市場真的就是像我們以為的人數多空比變化定義去走嗎??，讓我們來實際複盤一下

實際複盤

假設我們把市場分成有方向性的市場(多或空) 與 盤整期

那麼複盤的結果是完全相反的

我們從上圖可以看到，在 BTC 一直拉升的過程中 (2024/2月)，人數多空比 一直在往下打，這種情況符合前面的定義

BTC在盤整期階段(2024/3月~4月)，有看到完全與定義相反的行為

也就是部份出現 人數多空比降低▼ –> 市場看空▼ 人數多空比上升▲ –> 市場看多▲ 的情況

整理一下複盤的結果，觀察 價 / 人數多空比(LS) 之間與的市場情緒變化:

方向多空比市場
趨勢減少看漲
趨勢增加看跌
盤整增加看漲
盤整減少看跌

為什麼盤整期為發生這樣的情況呢?

行情始於分歧，終於一致，也就是買在分歧，賣在一致。

當市場一片看好，多空比降到 0.8 左右，當市場沒有新的資金動能再持續入注，價格自然就會下跌

當市場一片看壞，多空升到到 3 左右，當市場沒有人退出(躺平不想賣)，價格自然就會上升

依經驗，當多空比降到或升到極值附近時特別容易出現插針的情況，出現行情反轉的概率比較大。

如果在市場都在懵逼狀態，顯然沒有一致看多或看空時，那多空比就有可能出現相反的情況

不幸的是市場大部份時間都是混沌不明的況，也就是市場八成以上的時間都是盤整期

多空比結合未持倉

讓我們在複習一下未平倉量的變化

未平倉量增加: 多空雙方都有人進場
未平倉量不變: 如果配合量縮情況就是大家不想玩，反之就是混沌不明，有人進有人出
未平倉量下降: 多空雙方都有人離場

下面是主觀是依定義出來的組合

持倉量多空比描述
增加下降主力做多 / 散戶偏空
增加上升主力做空 / 散戶偏多
不變上升主力平多 / 散戶接手做多
不變下降散戶平多 / 主力接手做多
下降上升主力平多 / 散戶做空爆倉
下降下降散戶平多 / 散戶做多爆倉

上面是在「趨勢」情況下所進行的沙盤推演，實務上還是會隨著情況而有所異動 !

特別是盤整期混沌不明，可能造成比空比是與預期的相反情況

One More Thing

觀察多空比不要只看絕對值，更多時候建議應看一段時間相對變化值，例如四個小時、八個小時的變化

多空比只是一個觀察散戶情緒指標，不要只看單一指標，應結合OI變化、訂單簿、資金流動等同步進行觀察與比對

另外， Binance 交易所裡面的組成可能有機構、大戶、散戶等，用人數多空比來觀察市場上所有散戶的行為與情緒似乎可能還欠些什麼。

Gate.io 交易所組成絕大多數都是散戶，是一個相當不錯的觀察點，其有一個隱藏的API可以同時觀察標的 多的人數 與 空的人數 絕對值，經由Gate與Binance交叉比對更可以確認目前散戶的情緒。
