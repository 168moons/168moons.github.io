資金控管

發表於2024-08-14|更新於2024-08-11|learn

加密市場的資金配置

加密貨幣市場是一個高風險高回報的投資領域，資金配置在這裡特別重要! 首先明確你的投資目標（如短期投機、長期持有、分散風險等）和風險承受能力。加密貨幣市場波動性極高，因此需要確保你能夠承受潛在的損失。

首先，先來分類

1、主流加密貨幣：如比特幣（BTC）、以太坊（ETH）。這些通常被視為相對穩定的選擇。
2、山寨幣（Altcoins）：Solana（SOL）等，這些幣種風險較高，但潛在回報也較高。
3、穩定幣（Stablecoins）：如USDT、USDC。這些幣種與法定貨幣掛鉤，風險較低，適合作為避險資產。

要如何制定資金配置策略呢? 要根據你的風險承受能力和投資目標

下面為常見的配置模型
1、保守型配置: 穩定幣70% / 主流 20% / 山寨 10%
2、平衡型配置: 穩定幣35% / 主流 35% / 山寨 30%
3、積極型配置: 穩定幣10% / 主流 20% / 山寨 70%

加密市場變化迅速，因此需要定期檢查和調整資金配置。建議每月或每季度進行一次檢查，確保你的投資組合仍然符合你的目標和風險承受能力。

在之前，記得別忘了先做一下問券哦，先判斷與確認自已是什麼類型的投資類型?

資金控管原則

資金控管是投資最重要的入門課，通過有效的資金控管策略，投資者可以在市場波動中保持穩定，最大化收益並最小化風險。
以下是資金控管的一些基本原則和策略：

風險承受能力：

確定你能夠承受的最大風險。這通常是你總資金的一個百分比，例如1%或2%。
這個數字應該是你在每筆交易中最多能夠承受的損失。

分散投資：

不要把所有資金投入到單一資產或單一交易中。
通過分散投資來降低風險，確保即使某一筆交易失敗，也不會對整體資金造成重大影響。
資金控管的概念是：「不要把全部的錢押在同 1 支、同 1 個價位上。」

設置止損點：

每筆交易都應該設置止損點，即當交易虧損達到某個預定水平時，自動平倉以止損。
止損點應該根據市場波動性和個人風險承受能力來設置。

盈虧比：

確保每筆交易的潛在收益大於潛在損失。通常建議盈虧比至少為2:1。
這意味著每筆交易的潛在收益應該至少是潛在損失的兩倍。

資金配置後執行策略

加密市場的資金配置共有三個種類， 穩定幣、主流、山寨，下面將簡單介紹一下不同資產對應的不同對策

穩定幣收入來源

1、可以將穩定幣存入各大中心化交易所(簡單賺幣)，賺取被動收入，牛市時期更可能超20%以上，熊市最低可能就1%。
2、除了簡單賺幣，還一種P2P 類型的交易所，例如Bitfinex，這重點在於在高利時期儘可能放貸出去(最長可以放120天)。
3、將穩定幣存入中心化借貸平台，如Nexo等，這些平台也會支付利息給存款人。請要非常注意借貸平台安全性!
4、將穩定幣提供給去中心化交易所（如Uniswap、SushiSwap、Balancer等）的流動性池，賺取交易手續費和流動性獎勵(流動性挖礦)。
5、穩定幣存入收益聚合器平台（如Yearn Finance），這些平台會自動將資金分配到不同的收益策略池中，以最大化收益。
6、利用不同交易所之間的價格差異進行套利。
7、利用不同穩定幣之間的價格差異進行套利。例如，當USDT的價格低於USDC時，可以買入USDT並兌換成USDC，賺取差價。

主流/山寨幣收入來源

主流幣要如何操作? 請參考本站其它篇文章，基本上有長期持有（HODLing）、通過價格波動賺取差價、利用期貨和合約交易進行槓桿操作
山寨幣，包括持有和增值、交易和套利、挖礦、質押、流動性挖礦、收益農耕、參與項目、代幣化資產和參與治理等。

由於加密貨幣市場是一個高風險高回報，建議應利用下面kelly 公式計算出最佳的投資比例，以最大化長期收益。

尋找最佳投資比率的工具 -kelly 公式

Kelly公式是一種資金管理策略，用於決定在每次投資或賭博中應該投入的資金比例，以最大化長期資本增長。簡單來說，對自己有利的時候下大注，對自己不利的時候下小注或不下注。

先確定勝率（p）和賠率（b）：

勝率（p）：這是你預測某次交易成功的概率。可以通過歷史數據分析、技術分析或其他方法來估算。
賠率（b）：這是每單位投入的潛在收益。例如，如果你預測某加密貨幣的價格會上漲10%，那麼賠率就是0.1。
失敗的概率（q）：等於1減去勝率（p），即 ( q = 1 - p )。
(f^*) 是應該投入資金的比例

Kelly公式： f= (b*p-q)/b = p-q/b

假設你在下注，賠率是3（即你下注1元，贏了可以得到3元），你認為你有50%的機會會贏（即 ( p = 0.5 )）。

首先，先計算輸的概率 ( q )： [ q = 1 - 0.5 = 0.5 ]

將數值代入簡化版的公式： f = 0.5－0.5/3 = 33%

那就代表應該投入資金的33.3% 將會有最大的收入

範例1：高勝率0.8、高賠率0.5

假設有一個加密貨幣交易機會，成功的概率（p）為80%（即0.8），失敗的概率（q）為20%（即0.2），賠率（b）為0.5（即預期收益為50%）。

f = 0.8－0.2/0.5 = 0.4 –> 這代表應該投入資金的40%來進行這筆交易。

範例2：中等勝率0.6、中等賠率0.3

成功的概率（p）為60%（即0.6），失敗的概率（q）為40%（即0.4），賠率（b）為0.3（即預期收益為30%）。

f = 0.6 - 0.4/0.3 = -0.733 –>這代表會賠錢不值的投資

範例3：低勝率0.3、高賠率2

成功的概率（p）為30%（即0.3），失敗的概率（q）為70%（即0.7），賠率（b）為2（即預期收益為200%）。

f = 0.3 - 0.7/2 = -0.05 –>這代表會賠錢不值的投資

範例4：高勝率0.9、低賠率0.1

成功的概率（p）為90%（即0.9），失敗的概率（q）為10%（即0.1），賠率（b）為0.1（即預期收益為10%）。

f = 0.9 - 0.1/0.1 = -0.1 –>這代表會賠錢不值的投資

看完了上述之後，是不是覺得投資太難了~

由於Kelly公式依賴於準確的勝率和賠率估計。在加密貨幣市場中，這些數據可能會因市場波動而變得不穩定，因此需要謹慎估算。

有些投資者會選擇使用「分散Kelly」策略，即只投入Kelly公式計算結果的一部分資金（例如50%），再進一步降低風險。

使用Kelly公式來評估多個投資機會，選擇那些具有較高預期收益和較低風險的項目。

Kelly公式的優勢在於長期收益最大化，因此需要有耐心，不要因短期波動而輕易改變策略。

資金加碼策略

DCA 定期定額投資法

這方法通常適合於長期投資，通過不斷買入資產來平均分攤成本。最重要的是，一旦制定了投資計畫，就要嚴守紀律，持續買進。，無論市場價格如何，每次都投資相同金額的資金。例如每月投資 1000 美元。

每次執行固定頻率與金額，也比較好掌握資產狀況，做理財規劃。但需要克服人性，面對市場的波動，我們心裡會有很多聲音在左右我們的決策，讓我們容易受到情緒影響而做出非理性的選擇。如果我們能承受波動，並願意長期持有，也看好後續加密貨幣市場的表現，那麼長期看好 BTC 是一個不錯資產配置選擇。

順勢。金字塔加碼法 (右側交易)

金字塔加碼法是一種逐步增加投資的策略，通常在初始投資成功後進行。這種方法的目的是在市場趨勢明確時，逐步增加投資，以最大化收益。

傑西·李佛摩認為，買賣股票或是其它金融商品想要真正賺到錢，就必須一進場就要立即獲利，所以尋找關鍵點就變得十分重要。

在價格上漲時，往上加碼的數量一定要依序遞減；

操作步驟：
初始投資：首先進行一筆初始投資，假設為10,000元。
市場確認：當市場價格上漲到一定程度（例如10%）時，確認市場趨勢。
第一次加碼：在確認市場趨勢後，增加一筆的投資，假設為4,000元。
再次確認：當市場價格再次上漲到一定程度（例如再上漲10%）時，再次確認市場趨勢。
第二次加碼：再增加一筆更小的投資，假設為3,000元。
其它依此類推
當價格由最高點往下到一定程度時，全部出場。

逆勢。金字塔加碼法 (左側交易)

逆勢交易是一種在市場趨勢反轉時進行交易的策略，即在市場下跌時買入，在市場上漲時賣出。需要投資者具備較強的市場分析能力和風險承受能力。

逆勢交易中的金字塔加碼法是一種高風險高回報的策略，這種方法需要嚴格的風險管理和市場分析，以確保在市場反轉時獲得最大收益。

當市場價格越來越不利時，逐步加碼，以期在市場反轉時獲得更大的收益。(又稱馬丁策略)

價格下跌時，往下加碼的數量則可以依序遞增。

操作步驟：
初始投資：首先進行一筆初始投資，假設為10,000元。
市場下跌：當市場價格下跌到一定程度（例如10%）時，觀察反轉訊號。
第一次加碼：反轉訊號條件沒有消失情況下，增加一筆較小的投資，假設為1,000元。
再次下跌：當市場價格再次下跌到一定程度（例如再下跌10%）時，再次確認反轉訊號條件沒有消失情況下。
第二次加碼：再增加一筆比較大的投資，假設為2,000元。
其它依此類推
當價格由最下往上推升到一定程度時，全部出場。

備註：這種馬丁策略勝率非常高，但如果運氣不好時，低接後又還有更低，是會一波帶走人的。

除了上述 正金字塔加碼方法之外，還有另一種 倒金字塔加碼方法。反金字塔加碼法與金字塔加碼法相反，是在初始投資成功後，逐步減少投資金額。這種方法的目的是在市場趨勢不明確時，逐步減少投資，以降低風險。

選擇哪種策略取決於你的風險承受能力和市場判斷能力。無論選擇哪種策略，風險管理都是至關重要的。

我心目中最佳的配置方式如下
定期再平衡：定期檢查和調整你的投資組合，確保資金分配仍然符合你的投資目標和風險承受能力。 請善用交易所的子帳號功能，將資產與帳號進行分類

由於加密貨幣隨著BTC，有固定的周期循環 (四年一次)

1、熊市:

經由穩定幣收入 –> 產出的「利息」，在熊市定期定額方式投入BTC、ETH較優質幣種資產

拿著不動，在大週期頂部時機出現時才會考慮賣出

2、牛市初期:

牛市初期往上沖的主角通常都是 BTC –> 邊拉升 BTC 過程中，逐步將 BTC 的部位轉移到高風險有潛力的資產，例如 SOL 。

把穩定幣產出的「利息」改成定期定額買入高風險資產幣種

一部份的利息進行短線交易 （做多為主），如果有比較好的開倉時機，請善用槓桿與合約

3、牛市末期:

牛市中後期的現象就是BTC已經拉不太動了，BTC.D 逐步在下降

此時都在漲小型的幣，散戶過度FOMO，是期現套利、跨期套利、等套利最好的黃金時期，這些套利，此時的年化應該都有30~50% 以上

將一個「小小小資金部位」放到滾倉戶頭，每次利潤達一定程度後要持續抽回本金，其它時候就看戲 看看會幣價飛到那裡?

在ALT Coin上沖的過程中逐步出清優質幣種BTC、ETH部位，最好是最高時出清全部?? –> 換回穩定幣

如果不想動BTC也行， 那就換個空方的思路。
1、把BTC做為本金，開啟交易所的跨幣種保證金模式(統一帳號)，也就是把BTC做為抵押品(抵價值的95%)
2、依此基礎上全面且分散的去空，特別是被高估的山寨幣(特別是VC幣)，我們要開空的價值不能超過抵押品的價值。
3、當山寨幣發生大規模被清算時先平空，利用賺到的USDT去買更多BTC做為保證金
4、山寨幣反彈到一定程度後，繼續空!
5、依2~4 逐步去做
那什麼時候可以不要空?? 牛市中期後，也就是BTC已經帶大家飛一小段，準備要交接棒小幣時。
