資金費率

定義

加密貨幣的永續合約期貨市場為了保持合約價格與標的資產價格之間的平衡而設定的費率，由於永續合約永遠不會到期，為了讓現貨與合約之間價差能夠接近，資金費率是用來調整合約的成本或收益。

當永續合約的價格偏離標的資產價格時，交易所將通過調整資金費率來促使多頭或空頭向反方向支付資金，從而使合約價格回歸標的資產價格。

公式

資金金額 = 標記價格 * 合約規模 (U 本位合約) * 資金費率

資金費率由兩個因素決定：利率和溢價。
溢價指數 (P) = [Max(0,衝擊買方出價-價格指數) - Max(0,價格指數-衝擊賣方出價)] / 價格指數

簡單而言，溢價指數衡量市場買賣雙方之間的差異情況，從而判斷市場的買賣壓力

資金費率 （F） = 平均溢價指數 （P） + clamp （利率 （I） - 平均溢價指數 （P）， 0.05%， -0.05%）*

如果 (I-P) 是在 +/-0.05% 之間，那麼 F = P + (I-P) = I。換句話說，資金費率將等於利率

注意事項

資金費率通常預設為每日0.03% (資金計算每8 小時執行一次，因此每個資金間隔利率為0.01％)，計算時間為東八區 (UTC+8)時間08:00、16:00和24:00，實際收付時間將在 1 分鐘內收付完成。其中特別的交易對 ETHBTC、BNB 的預設基礎利率為 0%。

有一些高風險的被認定為市場操縱幣種(低市值的山寨幣)會調整收取的時間，可能為四小時或二小時收取一次資金費，且資金費率上下限將被調高為 3%及 -3%，詳細資料請參考 合約公告 🔗

主流的交易所預設是每八小時收取一次資金費率，但有一些交易所例如 Kraken 等是每一個小時都在收取費率，具體的規則要視各交易所的規定。

主流的交易所都有提供 預測資金費率，預測費率是每分鐘隨時在變動的，實務上也可以用現貨-永續的價差簡單快速去推算，最後每八小時實際收取的費率不完全與預測費率相同。

資金費率是永續合約最重要的調節期現價格機制，如果有交易所宣稱不需要支付資金費率，那要特別小心，例如之前搞詐騙的JPEX交易所。

觀察情緒變化

資金費率是重要的散戶情緒觀察指標，正資金費率通常表明市場上多頭較為強勢，而負資金費率則表明空頭較為強勢。

市場看漲時，資金費率通常為正。多頭倉位交易者會付錢給空頭交易者。
市場看跌時資金費率通常為負，空頭倉位交易者會付錢給多頭倉位交易者。

為了貼近取得目前市場散的情緒，我們收錄了主流的交易所資金費率資料，共挑選下列幣對做為代表全市場目前的情緒
‘BTC’,’ETH’,’SOL’,’XRP’,’DOGE’,’LINK’,’UNI’,’ADA’ ,’MATIC’,’LTC’,’ARB’,’OP’,’AVAX’,’ATOM’,’DOT’,’FIL’,’EOS’

交易所代號分別對應為B:Binance,O:OKX,Y:Bybit,K:Kucoin,I:Bitmex,T:Bitget,X:BingX,E:CoinEx,G:gate.io,H:HTX

同時間可以看出不同交易所資費分佈情況

其中分別針對三大所(Binance,OKX,Bybit) 去計算平均費率
1、BE (BTC+ETH) : 用來衡量市值最高的幣
2、ADFE(EOS+DOGE+FIL+ADA) : 散戶中的代表(老幣) ，例如 EOS 套了一堆人，更是有名的末日戰車

為了直觀的了解，利用顏色方式去呈現目前市場現況，愈紅代表市場越多人在看多，越紅越FOMO
相反的，如果越綠代表市場越悲觀，越綠代表越絕望。

資費情緒指數

為了更有效率且直觀顯示目前市場的資費情緒狀況，我們可以用下列方式先來算出資費河流圖

1、取得主流交易所進行OI加權，取得平均加權資費
2、只取同時在四家以上有上架的交易對，目前共有300個左右
3、分別依加權平均資費，上與下各分成五個等級，算出佔比
U1 (>0.015 )、U2 (>0.02 )、 U3 (>0.025 )、U4 (>0.03 )、U5 (>0.05 )、
D1 (<-0.005)、D2 (<-0.01 )、 D3 (<-0.015 )、D4 (<-0.02 )、D5 (<-0.05 )
4、資費情緒指數(FundingRateFlow FRF)
U1 + U2 + U3 + U4 + U5 - 2* (D1 +D2 +D3+D4 +D5)
簡單計算河流圖後，並把它轉成一個可以視覺化的指數衡量目前散戶的情緒
